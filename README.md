📘 Project Overview

eTuitionBd is a full-stack modern Tuition Management Platform designed for students, tutors, and administrators.
The platform streamlines the entire tuition process — from posting tuition requirements, applying for tuition, approving tutors, making secure payments, and managing all activities through a dedicated admin panel.

The main goal of this project is to provide a trusted, automated, and transparent way for students to find qualified tutors and for tutors to connect with verified tuition opportunities.

🎯 Purpose of the Project

....... To solve the real-world problem of finding reliable tutors and verified tuition posts.

....... To automate the workflow of tuition posting, tutor applications, approvals, and payments.

....... To ensure a secure system using role-based authentication and JWT verification.

....... To help admins monitor platform activities, manage users, and track platform earnings.

**This project is a Bangladeshi tuition project. There may be some additional features missing for this**

⭐ Key Features
🔵 Student Features

- **Register & log in with Firebase + JWT**

- **Post, update, and delete tuition posts**

- **View tutor applications**

- **Approve tutors with secure Stripe payment**

- **Manage profile and payment history**

### 🟢 Tutor Features
- **Browse all available tuition posts**
- **Apply to tuition with expected salary, experience, and qualifications**
- **Track application status**
- **Manage ongoing tuitions and revenue history**

### 🔴 Admin Features
- **Manage all users** (update, delete, role change)
- **Approve or reject tuition posts**
- **View all successful transactions**
- **Analyze earnings through reports & analytics**

### 🎨 UI & UX
- **Modern responsive UI** using Tailwind + DaisyUI
- **Smooth animations** with Framer Motion / GSAP
- **Clean dashboard layouts**
- **Error page & loading screen** included
